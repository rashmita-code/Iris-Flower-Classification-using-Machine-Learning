\# 🌸 Iris Flower Classification using Machine Learning



\## 📌 Overview

This project focuses on building a machine learning model to classify iris flowers into three species — Setosa, Versicolor, and Virginica — based on their physical measurements. It demonstrates the complete ML workflow from data analysis to model evaluation.



\---



\## 🎯 Objectives

\- Load and explore the dataset  

\- Perform data preprocessing  

\- Visualize feature relationships  

\- Train classification models  

\- Evaluate model performance  



\---



\## 📂 Dataset Information

\- File: `iris.csv`  



\### 🔑 Features:

\- Sepal Length  

\- Sepal Width  

\- Petal Length  

\- Petal Width  



\### 🎯 Target:

\- Species:

&#x20; - Setosa  

&#x20; - Versicolor  

&#x20; - Virginica  



\---



\## ⚙️ Technologies Used

\- Python 🐍  

\- Jupyter Notebook 📓  

\- Pandas \& NumPy  

\- Matplotlib \& Seaborn  

\- Scikit-learn

\## 🔄 Project Workflow



\### 1. Data Loading

\- Import dataset using Pandas  

\- Check structure and data types  



\### 2. Data Preprocessing

\- Handle missing values (if any)  

\- Encode categorical labels  



\### 3. Exploratory Data Analysis (EDA)

\- Pairplots for feature relationships  

\- Distribution plots  

\- Correlation analysis  



\### 4. Model Building

\- Implement classification algorithms:

&#x20; - Logistic Regression  

&#x20; - K-Nearest Neighbors  

&#x20; - Decision Tree  



\### 5. Model Evaluation

\- Accuracy score  

\- Confusion matrix  

\- Classification report

\- High accuracy achieved across models
